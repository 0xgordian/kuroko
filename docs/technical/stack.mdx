## Design system

Brutalist terminal aesthetic:
- Zero border-radius everywhere
- `#ff4500` brand orange
- `#0d0d0d` page background
- `#111111` panel background
- Geist Mono for all labels, numbers, badges
- Geist Sans for body copy
- `panel-bracket` — 3px orange left accent bar on all panels
