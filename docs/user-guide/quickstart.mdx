## Try it instantly

No setup required. Open the app and start chatting with the AI.

<Steps>
  <Step title="Open the app">
    Navigate to [kuroko.vercel.app](https://kuroko.vercel.app). The AI chat loads immediately.
  </Step>
  <Step title="Ask about markets">
    Type anything: "What markets are moving today?" or "Find me a good trade" or just press `/` to see all available commands.
  </Step>
  <Step title="Review the trade card">
    When the AI recommends a trade, a trade card appears inline. Review the market, side, shares, price, and reasoning.
  </Step>
  <Step title="Simulate or execute">
    Click **Simulate** to paper trade (no wallet needed) or **Execute** to place a real order (requires wallet).
  </Step>
</Steps>

## Slash commands

Type `/` in the chat input to open the command palette:

| Command | What it does |
|---------|-------------|
| `/trade` | Find the best trade opportunity right now |
| `/edge` | Score all markets and rank top 3 by edge |
| `/movers` | Biggest 24h probability movers |
| `/positions` | Review your open positions and P&L |
| `/alert` | Create a price alert |
| `/guard` | Set a stop-loss or take-profit rule |
| `/execute` | Go to the order terminal |
| `/portfolio` | Go to your portfolio |

## Voice input

Click the microphone icon in the chat input. Speak your query — it transcribes and sends automatically when you stop talking.
